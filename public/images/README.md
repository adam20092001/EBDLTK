Coloca aqui las fotos del restaurante para usarlas en la web.

Ejemplos de nombres recomendados:

- logo.png
- hero.jpg
- seco-cabrito.jpg
- arroz-con-pato.jpg
- ceviche.jpg
- seco-chavelo.jpg
- tamalitos-verdes.jpg
- duo-marino.jpg

Luego cambia las rutas en `data/site.ts`, por ejemplo:

```ts
logo: {
  src: "/images/logo.png",
  alt: "Logo de El Bunker de la Tia Karlita",
  initials: "TK"
}
```
